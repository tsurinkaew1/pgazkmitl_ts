About PGAz

Power Grid Analyzer (PGAz) offers a flexible and modern solution to overcome the limitations of existing power system analysis software, which provides advanced modeling capabilities and dynamic updates. As an open-source MATLAB-based tool, PGAz provides an accessible platform designed for analyzing and controlling future power grids. Now, PGAz offers four core features:

  1. Power Flow (PF) Analysis
  2. Optimal Power Flow (OPF) Analysis
  3. Small-Signal Stability Analysis (SSSA)
  4. Time-Domain Simulation (TS)
